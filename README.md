Jornada SQL: Da Base ao Avançado
Este repositório é um diário de aprendizado técnico. Aqui registro minha evolução em SQL, uma ferramenta essencial que estou integrando à minha bagagem de 10 anos na indústria da beleza para potencializar minhas análises de dados.

Estrutura do Repositório

Foco em CREATE TABLE e relacionamentos com FOREIGN KEY.

Próximos Passos: Consultas complexas (JOINS), agregações e análise de performance.
